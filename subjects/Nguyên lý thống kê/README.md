# Nguyên lý thống kê

> Môn học đã có lecture MD, 6 file luyện tập trắc nghiệm MD và 1 file bài kiểm tra MD.

Việc còn thiếu:

1. OCR lại file bài kiểm tra đang có `ocr_needed: true`.
2. Sinh lời giải trong `solutions/`.
3. Tạo ví dụ mở rộng trong `extensions/`.
4. Review độc lập sau khi có lời giải.
