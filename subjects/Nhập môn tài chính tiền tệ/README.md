# Nhập môn tài chính tiền tệ

> Môn học đã có lecture MD.

Việc còn thiếu:

1. Kiểm tra các đoạn lecture có `ocr_needed: true` và các flag OCR trong file MD.
2. Bổ sung đề bài vào `exercises/pdf/` hoặc `exercises/md/`.
3. Sinh lời giải trong `solutions/` sau khi có đề.
4. Tạo ví dụ mở rộng trong `extensions/`.
5. Review độc lập sau khi có lời giải.
