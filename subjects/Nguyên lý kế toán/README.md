# Nguyên lý kế toán

> Môn học đã có lecture MD và 7 file bài tập MD, nhưng các file bài tập hiện chủ yếu là ảnh scan.

Việc còn thiếu:

1. Chạy OCR lại các file bài tập đang có `ocr_needed: true`.
2. Convert/kiểm tra file đáp án bài tập thực hành trong `lectures/pdf/`.
3. Sinh lời giải trong `solutions/`.
4. Tạo ví dụ mở rộng trong `extensions/`.
5. Review độc lập sau khi có lời giải.
